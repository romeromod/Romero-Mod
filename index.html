Romero-Mod by romeromod

A Left 4 Dead 2 total conversion mod

Romero Mod is a total conversion mod for Left 4 Dead 2, inspired by George A. Romero's masterpiece zombie films. The mod has the following goals: * Evoke a feeling of genuine dread * Improve game world immersion * Dramatically change the strategies required to win * Create something fun!

Dependencies

Left 4 Dead 2

Install

It is recommended that you disable all other L4D2 add-ons when using the Romero Mod. Conflicts will almost certainly occur when other mods are present! Place the romeromod.vpk in the following folder (your path may vary slightly, depending on install location): C:\Program Files (x86)\Steam\steamapps\common\left 4 dead 2\left4dead2\addons Launch L4D2, and enjoy!
